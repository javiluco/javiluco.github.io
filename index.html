Portafolio de Título

CRVICE es un servicio web en el cual los usuarios pueden publicar una amplia variedad de servicios, ya sea en el ámbito profesional, académico o casual. Los usuarios pueden comunicarse y llegar a un acuerdo en un entorno privado, gracias a un chat integrado en la aplicación.


En cuanto a la infraestructura y el almacenamiento de datos, se utilizó Firebase, como Cloud Firestore para almacenar y recuperar datos en tiempo real, y Firebase Storage para el almacenamiento de archivos. Para el desarrollo de la interfaz de usuario, se empleó el framework Angular. Además, se utilizaron diversas librerías de Angular, como hot-toast para notificaciones, leaflet para la integración de mapas, ngx-paypal para pagos, forms para formularios y animations para efectos visuales. Cabe mencionar que todo el código del proyecto se desarrolló utilizando Typescript.

ClimaApp

ClimaApp es una aplicación web que permite a los usuarios consultar el clima actual y las previsiones meteorológicas en diversas ubicaciones. La interfaz de usuario está diseñada utilizando HTML y CSS. La funcionalidad está implementada en JavaScript, que se encarga de realizar solicitudes a una API externa para obtener datos meteorológicos en tiempo real.

CalendarApp

CalendarApp es una aplicación web diseñada para facilitar la planificación y gestión de eventos en grupo. Los usuarios pueden crear, editar, eliminar y compartir eventos, así como visualizar el calendario de manera colaborativa.
La interfaz se desarrolla con React y Bootstrap, garantizando una experiencia de usuario atractiva. La gestión del estado de la aplicación se maneja con Redux, lo que permite una actualización eficiente y fluida de la información. En el backend, se utiliza Node.js junto con MongoDB para el almacenamiento y recuperación de datos, asegurando una base de datos robusta y escalable. La comunicación entre el frontend y el backend se realiza a través de API RESTful, lo que permite una integración fluida y en tiempo real de los eventos y cambios en el calendario.

JournalApp

JournalApp es una aplicación web que permite a los usuarios llevar un diario personal de manera sencilla y atractiva. Los usuarios pueden crear, editar y eliminar entradas de diario, organizándolas de forma cronológica.
La interfaz está diseñada utilizando React y MaterialUI, lo que proporciona un diseño moderno y responsivo, así como una experiencia de usuario intuitiva.
La gestión del estado de la aplicación se realiza con Redux, lo que garantiza un flujo de datos eficiente y fácil de seguir. Para el almacenamiento de datos, se utiliza Firebase, que permite la sincronización en tiempo real de las entradas y asegura que los usuarios puedan acceder a sus diarios desde cualquier dispositivo. La integración de Firebase también simplifica la autenticación de usuarios, proporcionando un entorno seguro para cada usuario.

CRM de clientes

Client CRM es una aplicación web diseñada para gestionar y organizar la información de clientes de manera eficiente. Los usuarios pueden agregar, editar y eliminar registros de clientes, así como visualizar detalles relevantes en una interfaz amigable.
La aplicación está construida con HTML y estilizada utilizando Tailwind CSS, lo que proporciona un diseño moderno y responsive.
La lógica de la aplicación se maneja con JavaScript, utilizando IndexedDB para el almacenamiento local de datos, lo que permite un acceso rápido y eficiente a la información incluso sin conexión a Internet. Esta solución asegura que los datos de los clientes estén siempre disponibles y se sincronicen correctamente.

GifApp

GifApp es una aplicación web que permite a los usuarios buscar y explorar una amplia variedad de GIFs animados. Construida con Angular y estilizada utilizando Bootstrap, la aplicación ofrece una interfaz intuitiva y responsiva, facilitando la navegación y la búsqueda de contenido divertido y entretenido.
La funcionalidad de búsqueda se implementa mediante una API externa, que proporciona acceso a una vasta biblioteca de GIFs. Los usuarios pueden realizar búsquedas personalizadas y ver resultados en tiempo real.
